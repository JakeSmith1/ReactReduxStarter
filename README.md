# ReactReduxStarter

Structure:
- Client and Server
    - Client: React, Redux, React Router, React Bootstrap, Redux Promise, Redux Dev Tools
      - styling: scss: modules, partials, mixins, assets
    - Server: Node and Express
- Webpack

To Run:
- “npm install”
- "npm start"

For Development Run:
- "webpack -w"
- in another tab: "nodemon server/server.js"
